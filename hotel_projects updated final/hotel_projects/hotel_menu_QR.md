# Hotel Digital Menu & Ordering System (Prototype)

## Vision

A lightweight digital ordering system that replaces printed menus and streamlines communication between customers, kitchen staff, and hotel managers.

---

## Customer Experience

✅ Scan QR code on the table

✅ View the digital menu on their phone

✅ Browse food categories and prices

✅ Add items to cart

✅ Place an order without waiting for a waiter

---

## Hotel Experience

✅ Receive orders instantly on the kitchen screen

✅ Real-time order status updates

✅ Manager receives Telegram notification for every new order

✅ Simple dashboard to monitor orders

✅ Easily update menu items and prices

---

## Technology Stack

### Frontend

* React
* Tailwind CSS

### Backend

* Node.js
* Express.js
* Socket.IO (Real-time communication)

### Database

* SQLite (Prototype)
* Stores:

  * Menu items
  * Orders
  * Order status

### Notifications

* Telegram Bot API
* Sends instant order notifications to the manager

---

## Prototype Workflow

Customer scans QR Code

↓

Digital menu opens

↓

Customer selects food

↓

Customer places order

↓

Order is saved in the database

↓

Kitchen screen updates instantly

↓

Manager receives Telegram notification

↓

Kitchen changes order status
(New → Preparing → Ready)

---

## Goal

Build a simple but professional prototype that demonstrates how hotels can:

* Reduce waiting time
* Eliminate printed menus
* Improve order accuracy
* Speed up kitchen communication
* Digitize restaurant operations

The prototype should be polished enough to present to hotel owners and demonstrate the value of adopting the full system.
